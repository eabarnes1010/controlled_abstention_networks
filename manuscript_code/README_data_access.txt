The synthetic climate data used in these papers is approximately 720Mb, saved as a .mat file. The data is from Mamalakis et al. (2021) - with citation given below. 

To access the data, go to the following doi: 10.5281/zenodo.5750196

Mamalakis, Antonios, Imme Ebert-Uphoff and Elizabeth A. Barnes: Neural Network Attribution Methods for Problems in Geoscience: A Novel Synthetic Benchmark Dataset, submitted to Environmental Data Science, 11/2021, preprint available https://arxiv.org/abs/2103.10005.
